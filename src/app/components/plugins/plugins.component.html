MetaCleaner for Jira

MetaCleaner for Jira is a tool that removes the metadata of the files that are uploaded as attachments to Jira issues.

Why MetaCleaner for Jira?

Attaching files to Issues is very useful but can also be dangerous.

When we upload a file, it usually has several details about our identity that can compromise our privacy.

How it works?

Every time a new file is uploaded the tool automatically removes the metadata of the file by replacing the file with a clean copy.

Only the metadata is removed, the content of the file is kept intact.
